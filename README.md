# Libro de Recetas - BackEnd 📖

Este es un proyecto personal creado con fines didacticos.

## Tecnologias 🔨

### 1. LENGUAJE Y ENTORNO

- Node.js: Entorno de ejecución de JavaScript fuera del navegador
- TypeScript: JavaScript tipado, para disminuir errores

### 2. FRAMEWORK PRINCIPAL

- Express.js: "Andamio" para construir aplicaciones web del lado del servidor

### 3. BASE DE DATOS

- MySQL: Sistema de base de datos relacional
- mysql2: "Traductor" entre Node.js y MySQL

### 4. AUTENTICACIÓN Y SEGURIDAD

- bcryptjs: Algoritmo para hashear contraseñas
- jsonwebtoken (JWT): Token digital que verifica identidad
- express-validator: Validador de datos de entrada

### 5. EMAILS

- mailersend: Servicio profesional de envío de emails
- handlebars: Motor de plantillas para emails HTML

### 6. SEGURIDAD Y MIDDLEWARES

- cors: Permisos de comunicación entre dominios diferentes
- helmet: Configurador de headers HTTP seguros
- express-rate-limit: Limitador de intentos de requests

### 7. VARIABLES DE ENTORNO

- dotenv: Cargador de variables desde archivo .env

### 8. DESARROLLO Y CALIDAD

- nodemon: Reiniciador automático del servidor
- ts-node: Ejecutor de TypeScript sin compilar
- @types/\*: Definiciones de tipos para librerías JavaScript

### 9. LOGGING

- winston o morgan: Librerías para registrar actividades

### 10. DOCUMENTACIÓN

- swagger-ui-express: Generador de documentación interactiva de API
